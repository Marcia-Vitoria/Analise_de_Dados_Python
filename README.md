Este projeto consiste em um notebook de análise exploratória de dados, desenvolvido em Python, com o uso de bibliotecas amplamente empregadas na área de Ciência de Dados.  
O estudo foi realizado no ambiente Google Colab, com base nos conhecimentos adquiridos durante uma imersão promovida pela Alura.

---

## 1. Tecnologias e Ferramentas
- Python 3.x  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Plotly  

---

## 2. Descrição do Dataset
O conjunto de dados utilizado é de domínio público originalmente retirado do Kaggle e apresenta informações relacionadas a área de tecnologia, contendo variáveis como ano de trabalho, nível de senioridade, tipo de contrato, cargo, regime de trabalho e remuneração.

---

## 3. Metodologia

O desenvolvimento do projeto foi organizado nas seguintes etapas:

### 3.1 Análise dos dados  
Inspeção inicial do conjunto de dados, verificação da estrutura, tipos das variáveis e estatísticas descritivas.

### 3.2 Preparação e limpeza dos dados  
Padronização de variáveis categóricas, renomeação de colunas, identificação e tratamento de valores ausentes, bem como adequação dos tipos de dados.

### 3.3 Visualização dos dados  
Construção de gráficos estatísticos e interativos para análise.

---

## 4. Procedimentos para Execução

### 4.1 Execução no Google Colab
1. Acessar o endereço https://colab.research.google.com  
2. Realizar autenticação com uma conta Google  
3. Selecionar a opção **Upload de notebook**  
4. Carregar o arquivo `dados_com_python.ipynb`  
5. Executar as células de forma sequencial para reprodução das análises

### 4.2 Execução em ambiente local
1. Instalar o Python 3.x  
2. Instalar as dependências necessárias:
```bash
pip install pandas numpy matplotlib seaborn plotly
```
